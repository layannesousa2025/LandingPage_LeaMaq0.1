# 🛠️ L&A Maq - Landing Page de Assistência Técnica

Este projeto é uma landing page moderna e responsiva desenvolvida para a **L&A Maq**, uma empresa especializada em assistência técnica de refrigeração e eletrodomésticos (geladeiras, freezers, máquinas de lavar, etc.).

O objetivo principal da página é converter visitantes em clientes, destacando os serviços oferecidos, os diferenciais da empresa e facilitando o contato direto via WhatsApp ou formulário de orçamento.

---

## 🚀 Demonstração

Você pode visualizar o projeto online (se hospedado) ou clonando este repositório e abrindo o arquivo `index.html`.

---

## ✨ Funcionalidades

- **Design Responsivo:** Adaptado para celulares, tablets e desktops.
- **Navegação Suave (Smooth Scroll):** Experiência de usuário fluida ao navegar pelas seções.
- **Animações de Revelação:** Elementos que surgem suavemente conforme o usuário faz o scroll (usando `Intersection Observer API`).
- **Integração com WhatsApp:** Botão flutuante e chamadas para ação (CTAs) linkados diretamente ao app.
- **Formulário de Contato Funcional:** Configurado com [FormSubmit](https://formsubmit.co/) para recebimento de e-mails.
- **Menu Mobile Interativo:** Menu hambúrguer otimizado para navegação em dispositivos móveis.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

O projeto foi construído utilizando tecnologias modernas de desenvolvimento web sem a necessidade de frameworks pesados, garantindo performance e carregamento rápido:

- **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML):** Estrutura semântica para melhor SEO.
- **[CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS):** Estilização customizada com variáveis CSS, Flexbox e CSS Grid.
- **[JavaScript (Vanilla)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript):** Lógica de animações, menu mobile e rolagem suave.
- **[Google Fonts](https://fonts.google.com/):** Tipografia moderna utilizando a fonte *Outfit*.
- **[Font Awesome](https://fontawesome.com/):** Biblioteca de ícones vetoriais.
- **[FormSubmit](https://formsubmit.co/):** Serviço para processamento de formulários sem backend próprio.
